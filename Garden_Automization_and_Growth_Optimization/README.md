# Garden Automization and Growth Optimization

This project automates garden care and optimizes plant growth using microcontrollers. It features automated watering, monitoring, and a web interface for status and control.

## Contents
- Garden_Automization_and_Growth_Optimization.ino — Main automation code
- website2.h — Web interface definitions
- README.md — This file

## Features
- Automated watering and monitoring
- Growth optimization algorithms
- Web interface (see `website2.h`)

## How to Use
1. Open `Garden_Automization_and_Growth_Optimization.ino` in the Arduino IDE.
2. Upload to your microcontroller.
3. Customize logic as needed for your garden setup.
