# Arduino UNO Mini Elevator

This folder contains the Arduino UNO version of the Mini Elevator Project. It demonstrates controlling a mini elevator mechanism and displaying status or sensor data on an OLED display.

## Contents
- ARDUINO_UNO.ino — Main Arduino UNO code
- README.md — This file

## Features
- Controls a mini elevator mechanism
- Displays status and sensor data on an OLED screen

## How to Use
1. Open `ARDUINO_UNO.ino` in the Arduino IDE.
2. Upload to your Arduino UNO board.
3. Connect the OLED display as described in the code comments.
