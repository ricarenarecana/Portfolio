# Mini Elevator Project

This folder contains the Mini Elevator Project, which demonstrates a mini elevator system using both ESP32 and Arduino UNO microcontrollers. The project features code for each platform and documentation for setup and usage.

## Subfolders
- **ARDUINO_UNO/**: Arduino UNO version
  - ARDUINO_UNO.ino — Main code for Arduino UNO
  - README.md — Details for Arduino UNO version
- **ESP32/**: ESP32 version
  - ESP32.ino — Main code for ESP32
  - README.md — Details for ESP32 version

## Features
- Controls a mini elevator mechanism
- Displays status and sensor data on OLED screens
- Example code for both ESP32 and Arduino UNO

## How to Use
1. Choose your platform (Arduino UNO or ESP32).
2. See the corresponding subfolder for code and wiring instructions.
3. Upload the code to your board and connect the hardware as described.
