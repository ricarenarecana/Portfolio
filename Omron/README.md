# Omron PLC Projects

This folder contains various Omron PLC (Programmable Logic Controller) project files, including CX-Programmer files and related options.

## Contents
- Sorting System (metal&non-metal).cxp — CX-Programmer project for sorting system
- PICK AND PLACE SET AND RESET.opt — Options/configuration for pick and place project
- eto-na (1).opt — Options/configuration for eto-na project

## How to Use
- Open `.cxp` files in Omron CX-Programmer to view or edit ladder logic and project configuration.
- Use `.opt` files for project-specific options or settings.

Refer to each project file in CX-Programmer for detailed logic and configuration.
